⚠ Gotchas
• Refs invalidées après navigation → refaire snapshot
• 1er appel ~3s (Chromium launch), ensuite ~100-200ms
• Auto-shutdown après 30min d'inactivité
• Toujours -i d'abord, ne pas deviner les sélecteurs
• Vérifier console après chaque action
• /review retourne DONE | DONE_WITH_CONCERNS | BLOCKED | NEEDS_CONTEXT
• /investigate s'arrête après 3 tentatives ratées
Sortie status (toutes les skills)
DONE — succès · DONE_WITH_CONCERNS — ok avec notes
BLOCKED — impossible · NEEDS_CONTEXT — info manquante

Philosophie
"Boil the Lake" — solutions complètes, pas de raccourcis.
Pas de MCP : plain HTTP, 0 token overhead, ~100ms latency.
Chaque erreur est actionable pour l'agent AI.
